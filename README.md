## Projects

### AI Surface Defect Detection System
*Internship Project · 2025*

Full-stack quality assurance system that replaces manual visual inspection with an
explainable AI pipeline. Fine-tuned ResNet-50 on the MVTec AD dataset across four
surface categories (tile, carpet, leather, wood). Served via FastAPI microservice behind
a Node.js/Express gateway with JWT auth, MongoDB logging, and a React/TypeScript dashboard
with real-time Grad-CAM heatmap visualization.

**Stack:** Python · PyTorch · FastAPI · Node.js · MongoDB Atlas · React · TypeScript · Tailwind CSS

[![Repo](https://img.shields.io/badge/Repository-181717?style=flat-square&logo=github&logoColor=white)](https://github.com/msaadirfan-max/ai-surface-defect-detection-system)

---

### CI/CD Build Failure Prediction
*MLOps · 2025*

MLOps pipeline that predicts CI/CD build failures before they happen. Fetched real GitHub
Actions data from 48 open-source repositories, trained LightGBM with Optuna hyperparameter
tuning, and calibrated to 72% recall and 75.18% accuracy. Served via FastAPI REST API with
SHAP explainability, containerized with Docker, and targeted for AWS deployment.

**Stack:** Python · LightGBM · FastAPI · Optuna · SHAP · Docker · AWS

[![Repo](https://img.shields.io/badge/Repository-181717?style=flat-square&logo=github&logoColor=white)](https://github.com/msaadirfan-max/cicd-failure-prediction-mlops)

### StayMate — Serverless Housing Platform
*AWS Cloud · 2025*

Scalable serverless housing platform built entirely on AWS infrastructure. Lambda functions
handle all backend logic through API Gateway, PostgreSQL manages relational data, React
frontend is hosted on S3, and file uploads use presigned URLs to bypass the server entirely.
Three AWS Academy certifications informed the architecture decisions.

**Stack:** AWS Lambda · API Gateway · Node.js · PostgreSQL · React · Amazon S3

[![Repo](https://img.shields.io/badge/Repository-181717?style=flat-square&logo=github&logoColor=white)](https://github.com/msaadirfan-max/stay-mate-serverless)
