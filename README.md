## 🚀 Featured Project

### 🔍 [AI-Powered Surface Defect Detection System](https://github.com/msaadirfan-max/ai-surface-defect-detection-system)
An automated visual quality inspection ecosystem that replaces manual manufacturing checks with a consistent, auditable, and explainable AI pipeline.
* **The AI Core:** Fine-tuned a **ResNet-50** model using PyTorch, resolving extreme class imbalances via `WeightedRandomSampler` and customized loss penalties.
* **Explainability:** Integrated a **FastAPI** microservice executing raw **Grad-CAM heatmaps** to give real-time visual explanations of model decisions.
* **The Architecture:** Scaled via a decoupled **Node.js/Express** gateway handling secure JWT authentication, rate limiting, and image-forwarding.
